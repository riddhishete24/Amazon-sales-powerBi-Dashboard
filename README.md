# Amazon Sales Analytics Dashboard

## 📊 Project Overview
This project is an interactive Power BI dashboard built using Amazon sales data to analyze sales performance, profitability, and product trends.

## 🔹 Key Features
- Total Sales, Orders, Quantity, Profit, Profit Margin KPIs
- Monthly sales trend analysis
- Category and sub-category insights
- Top 10 products analysis
- Interactive slicers (Category, Date, State)

## 🔹 Tools Used
- Power BI
- DAX
- Excel / CSV dataset

## 📈 Outcome
This dashboard helps understand business performance, identify top-performing products, and support data-driven decision making.
